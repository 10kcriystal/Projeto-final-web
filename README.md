# Projeto web, e-comerce;

Projeto web desenvolvido em HTML e PHP como parte do meu estudo/prática em desenvolvimento web.
(Ainda sem responsividade para diferentes resoluções, em desenvolvimento)

## Tecnologias utilizadas
- HTML
- CSS
- PHP
- MySQL

## Requisitos 
- Servidor web (XAMPP)

## Como executar o projeto localmente
1. Clonar o repositório:
git clone https://github.com/10kcriystal/Projeto-final-web.git

2. Colocar o projeto na pasta do servidor web  
(ex.: `htdocs` no XAMPP)

## Funcionalidades
- Homepage
- Catálogo
- Produto
- Carrinho
- Checkout
